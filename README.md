# Smart Resume Analyzer 

An AI-powered Resume Analyzer built using Python and Streamlit that evaluates resumes based on role-specific skills, keyword matching, and ATS-style scoring.

## 🔗 Live Demo
https://smart-resume-analyzer-iw75ryd5hpe76fmjk27xna.streamlit.app/

---

# 📌 Features

- Upload Resume in PDF/DOCX format
- Role-based Resume Analysis
- ATS-style Match Scoring
- Skill Detection System
- Resume Improvement Suggestions
- Interactive Dashboard using Plotly
- Fast and Lightweight Streamlit UI

---

# 🛠️ Tech Stack

- Python
- Streamlit

# ▶️ Installation & Run Locally

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Resume-Analyzer.git
```

## 2️⃣ Open Project Folder

```bash
cd Smart-Resume-Analyzer
```

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 4️⃣ Run Streamlit App

```bash
streamlit run app.py
```

---

# 📊 How It Works

1. User uploads resume
2. Resume text is extracted
3. Skills are matched against selected role
4. ATS-style score is calculated
5. Suggestions are generated
